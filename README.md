# russian-text-summarization

Проект для автоматического суммаризации русскоязычных текстов и построения кратких рефератов на основе исходной статьи.

**Описание работы проекта приведено в отчете:** [отчет](https://docs.google.com/document/d/1dosjgTRXzqhvPCP3h3wysnQ9QU1RJd2MEYaWYhC7dB0/edit?tab=t.0)

## Описание

Проект решает задачу автоматического сокращения текста с сохранением его основной смысловой нагрузки.

Цель работы — получить краткий и связный реферат исходной статьи за счет:
- предварительной нормализации текста;
- лемматизации и фильтрации стоп-слов;
- выделения наиболее значимых предложений;
- формирования итоговой краткой сводки текста.

## Архитектура

Проект состоит из нескольких основных этапов:

### 1. Загрузка и подготовка данных
В качестве источника используется датасет `IlyaGusev/gazeta`.

На этом этапе:
- загружаются статьи и рефераты;
- выполняется нормализация текста;
- текст разбивается на предложения;
- проводится лемматизация;
- удаляются стоп-слова.

### 2. Векторизация и ранжирование предложений
Для оценки значимости предложений используются:
- TF-IDF-векторизация;
- семантическая близость предложений;
- MMR-подход для выбора разнообразных и информативных фрагментов.

### 3. Формирование итогового реферата
Суммаризация строится как набор наиболее важных предложений, ограниченный заданной длиной.

## Методы и метрики

Для оценки качества суммаризации используются:
- ROUGE-1
- ROUGE-2
- ROUGE-L
- BERTScore

Метрики позволяют сравнивать сгенерированный реферат с эталонным.

## Структура проекта

- `main.ipynb` — основной ноутбук с реализацией алгоритма суммаризации.
- `README.md` — описание проекта.

## Использование

Основной сценарий работы:
1. загрузить данные;
2. выполнить нормализацию и лемматизацию;
3. разбить текст на предложения;
4. оценить важность предложений;
5. собрать итоговый реферат;
6. проверить качество с помощью метрик.
